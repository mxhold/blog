2016 May 15

I made this post to test my blog's ability to do syntax highlighting.

Here's some Ruby:

module Greeting
  def self.hello(arg)
    puts "Hello, #{arg}!"
  end
end
Greeting.hello("World") # => "Hello, World!"

Here's some Elixir:

defmodule Greeting do
  def hello(arg) do
    IO.puts "Hello, #{arg}!"
  end
end
Greeting.hello("World") # => "Hello, World!"

Here's some Rust:

mod greeting {
    pub fn hello(arg: &str) {
        println!("Hello, {}!", arg);
    }
}
fn main() {
    greeting::hello("World"); // => "Hello, World!"
}
